A simple game thrown together in a couple weeks. Made by me and a couple friends as we were bored during the covid-19 lockdown.

Here you can have a fullscreen version if you like it more or if your browser is confused.

This game uses an aset pack from opengameart.org by Kenney.nl

A little explanation

The game can be controlled with WASD or the arrow keys. The only objective is to reach the orange worm at the end. In "Chad" mode you only get a single checkpoint about half way through, but you can choose to avoid that as well. The game *can* be finished even if it doesn't seem so.

Be ready for performance issues in the second half as the game is poorly optimized.
